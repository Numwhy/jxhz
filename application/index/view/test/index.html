检测项目

型式试验

型式试验项目是特定产品 按标准规定，在送试样品上进行的一系列试验。

凝露试验

试验依据主要是DL/T539。试验名称为：凝露条件下的绝缘试验。内容为：常温凝露条件下的工频、冲击耐受试验。

高原试验

海拔超过1000米环境中，测试由于海拔升高，对试验样品的电气特性的影响。

EMC电磁兼容

在其电磁环境中符合要求运行并不对其环境中的任何设备产生无法忍受的电磁干扰的能力

人工污秽试验

按规定条件，使试品表面受到人工污染和充分湿润，并对试品施加高电压的试验。

抗震试验

通过在模拟地震振动台上的地震考核试验，观察样机在地震荷载作用时和作用后的功能，观察样机有无变形等机械损伤，并分析试验结果，鉴定试件抗震能力。
